9. ⭐ 温度変換プログラム 🌡️#

# (°C x 9/5) + 32 = °F
# (°F - 32) x 5/9 = °C
# °(Option + Shift + 8)
unit = input("この温度は摂氏または華氏ですか？ (C/F): ")
temp = float(input("温度を入力してください: "))

if unit == "C":
    temp = round((9 * temp) / 5 + 32, 1)
    print(f"華氏での温度は: {temp}°Fです")
elif unit == "F":
    temp = round((temp - 32) * 5 / 9, 1)
    print(f"摂氏での温度は: {temp}°Cです")
else:
    print(f"{unit} は有効な測定単位ではありません")

14. フォーマット指定子 💬#

{value:flags} は、挿入されたフラグに基づいて値をフォーマットします

  • .(数値)f = 指定された小数点以下の桁数に丸める（固定小数点）
  • :(数値) = 指定された数のスペースを割り当てる
  • :03 = 指定された数のスペースを割り当て、ゼロで埋める
  • :< = 左揃え
  • :> = 右揃え
  • :^ = 中央揃え
  • :+ = 正の値にプラス記号を使用する
  • := = 符号を最左の位置に配置する
  • : = 正の数の前にスペースを挿入する
  • :, = カンマ区切り
