★ 第一部分：句子核心与基础粒子 (Lessons 1-20)

这部分奠定了整个课程的基础，讲解了日语句子的基本构成、核心助词 が、は、も、の 的用法，以及地点、时间和工具的表示方法。

◇日语超基础词汇表
#

本词汇表收录了日语学习中最基础的词汇，适合初学者用于构建简单的"A是B"和"A做B"句型。所有词汇均标注平假名、罗马音和中英文含义。

  1. Nouns

    1. うさぎ (usagi) = rabbit 兔子
    2. ひと (hito) = person 人
    3. こ (ko) = child 孩子
    4. とり (tori) = bird 鸟
    5. おんな の ひと (onna no hito) = woman 女人
    6. おんな の こ (onna no ko) = girl 女孩
    7. おとこ の ひと (otoko no hito) = man 男人
    8. おとこ の こ (otoko no ko) = boy 男孩
    9. にほんじん (Nihonjin) = Japanese person 日本人
    10. あめりかじん (Amerikajin) = American person 美国人
    11. ふらんすじん (Furansujin) = French person 法国人
    12. ぱん (pan) = bread (same as Spanish) 面包

  2. (Free gifts!)

    1. ぺん (pen) = pen 钢笔
    2. あいすくりーむ (aisu kuriimu) = ice cream 冰淇淋
    3. たくしー (takushii) = taxi 出租车
    4. けーき (keeki) = cake 蛋糕

  3. Verbs

    1. あるく (aruku) = walk 走路，步行
    2. とぶ (tobu) = jump/fly 跳跃/飞行
    3. うたう (utau) = sing 唱歌

  4. Adjectives

    1. あかい (akai) = red 红色的
    2. あおい (aoi) = blue 蓝色的
    3. うれしい (ureshii) = happy 高兴的，愉快的
    4. おいしい (oishii) = tasty, delicious 好吃的，美味的
    5. おもしろい (omoshiroi) = interesting, amusing 有趣的，好玩的

  5. 学习建议

    1. 优先使用平假名：初学者应重点掌握平假名写法
    2. 循序渐进：掌握基础词汇后再学习汉字和片假名写法
    3. 实际应用：尝试用这些词汇构造简单句子进行练习
    4. 发音练习：借助罗马音标注练习正确发音

◇ [1-2] 核心句子结构
#

  1. 核心概念：句子火车 (Sentence Train)
    1. 所有日语句子都由 A (主语/主格，Main Carriage) 和 B (谓语，Engine) 构成。
    2. A が B 是句子的最基本形态。が 是连接主语和谓语的“连接销”。
  2. 三种基本句型:
    1. A does B (动词句): A が [动词（以u结尾）]
      1. 例: さくら が あるく (Sakura ga aruku) - 樱走。
    2. A is B (名词句): A が [名词] だ
      1. だ (da) 是判断助动词，相当于“是”。
      2. 例: さくら が 日本人(にほんじん) だ (Sakura ga Nihonjin da) - 樱是日本人。
    3. A is B (形容词句): A が [い-形容词]
      1. い-形容词 本身包含“是”的状态，无需加 だ。
      2. 例: ペン が あかい (Pen ga akai) - 笔是红的。
  3. 核心概念：隐形主语 (Invisible Carriage / Zero Pronoun)
    1. 当上下文清晰时，主语 A が 会被省略，但它在逻辑上仍然存在。这是日语口语和书面语极其常见的现象。
    2. 例: ケーキ を 食べる (keeki o taberu) - 吃蛋糕。(主语“我”或“你”被省略)
  4. 助词 を (o): 标记动词的直接宾语（作用对象）。
    1. 例: 私 が ケーキ を 食べる (watashi ga keeki o taberu) - 我吃蛋糕。
    2. 是句子逻辑结构的一部分，不是核心列车的一部分。核心列车始终仅由两个元素组成，即引擎和主车厢。
    3. 它们在告诉我们有关引擎（谓语）或主车厢（主语）的更多情况。

◇ [3-5] 核心助词 は (wa) 与 も (mo)
#

  1. 助词 は (wa): 话题标记 (Topic Marker)
    1. は 是一个非逻辑助词，它是一面旗帜，标记的是“我们正在谈论的东西”，它将一个元素从句子中提取出来，作为上下文的背景或主题。
    2. は 会覆盖 が 和 を，但不会覆盖其他助词。
    3. X は Y が Z 结构非常常见，意为 “关于X，是Y做了Z”。
    4. 例: 象(ぞう) は 鼻(はな) が 長(なが)い (zou wa hana ga nagai) - 大象，鼻子长。 (谈论的是大象，核心句子是“鼻子长”)
  2. 助词 も (mo): “也” (Also/Too)
    1. も 表示与前述情况相同、类似。
    2. 它同样会覆盖 が 和 を。
    3. 例: 私 が 行く。あなた も 行く。 (Watashi ga iku. Anata mo iku.) - 我去。你也去。
    4. 例: ケーキ を 食べる。クッキー も 食べる。 (Keeki o taberu. Kukkii mo taberu.) - 吃蛋糕。也吃曲奇。

◇ [6-10] 助词 の (no): 所属与属性
#

  1. 助词 の (no): 所属关系标记
    1. 最常见的用法是表示“的”，连接两个名词 X の Y (Y of X)。
    2. 例: 私 の 猫 (watashi no neko) - 我的猫。
  2. の 的名词化功能:
    1. の 可以将前面的动词或形容词短语变成一个名词性的“东西”或“那个”。
    2. 例: 赤い の が 好き (akai no ga suki) - 喜欢红色的（东西）。
  3. の 替代 が: 在从句中，の 可以用来替代 が，使语气更柔和或更具解释性。
    1. 例: 私 が 行く (watashi ga iku) -> 私 の 行く... (watashi no iku…) - 我去的（地方）…

◇ [11-20] 地点、方向与工具助词
#

  1. 助词 に (ni):
    1. 静态存在地点: 与 いる (iru) / ある (aru) (在) 连用，表示存在的地点。
      1. 例: 猫 が 箱(はこ) に いる (neko ga hako ni iru) - 猫在箱子里。
    2. 动作归着点/目的地: 表示动作的朝向或终点。
      1. 例: 東京 に 行く (Tokyo ni iku) - 去东京。
    3. 具体时间点: 表示动作发生的特定时间。
      1. 例: 三時 に 会う (san-ji ni au) - 三点见。
  2. 助词 へ (e): 方向标记
    1. 与 に 相似，但更强调“朝…方向去”，不一定非要到达。
    2. 例: 東京 へ 行く (Tokyo e iku) - 朝东京方向去。
  3. 助词 で (de):
    1. 动态动作发生地: 表示动作发生的场所。
      1. 例: レストラン で 食べる (resutoran de taberu) - 在餐厅吃。
    2. 方式/工具/材料: 表示完成动作所用的方法或工具。
      1. 例: バス で 行く (basu de iku) - 坐公交去。
      2. 例: 木 で 作る (ki de tsukuru) - 用木头做。
  4. 助词 から (kara) & まで (made):
    1. から: from, “从…开始”。
    2. まで: until, “到…为止”。
    3. 可以用于时间、地点、范围。
    4. 例: 家 から 学校 まで (ie kara gakkou made) - 从家到学校。

★ 第二部分：动词、形容词与时态 (Lessons 21-45)
#

这部分深入讲解了日语两大核心词类——动词和形容词的分类、变形（活用）以及如何表达时态和否定。

◇ [21-30] 动词分类与 て-形
#

  • 动词分类:
    1. る-动词 (ru-verb / Group 2): 词尾是 る，且 る 前面的元音是 i 或 e。如 食べる (taberu)、見る (miru)。
    2. う-动词 (u-verb / Group 1): 其他所有动词。如 歩く (aruku)、話す (hanasu)、待つ (matsu)。
    3. 不规则动词: する (suru) 和 来る (kuru)。
  • 核心变形：て-形 (Te-form)
    • て-形 是动词最重要的变形，用于连接句子、表达请求、许可、正在进行等。
    • 变形规则:
      • る-动词: 去掉 る + て。 食べる -> 食べて。
      • う-动词: 规则较为复杂，根据词尾变化（音便）。
        • く/ぐ -> いて/いで: 歩く -> 歩いて
        • う/つ/る -> って: 待つ -> 待って
        • ぬ/ぶ/む -> んで: 飲む -> 飲んで
        • す -> して: 話す -> 話して
      • 不规则: する -> して，来る -> 来て (kite)。
  • て-形 的用法:
    1. 连接动作 (and): 起きて、ご飯を食べて、学校へ行く - 起床，然后吃饭，然后去学校。
    2. 请求 (てください): 待ってください (matte kudasai) - 请等一下。
    3. 正在进行 (ている): 食べている (tabete iru) - 正在吃。
    4. 原因/方式: 歩いて行く (aruite iku) - 走着去。

◇ [31-40] 形容词分类与变形
#

  • い-形容词 (i-adjectives): 以 い 结尾。
    • 否定: い -> くない: 赤い (akai) -> 赤くない (akakunai)。
    • 过去式: い -> かった: 赤い -> 赤かった (akakatta)。
    • 过去否定: い -> くなかった: 赤い -> 赤くなかった (akakunakatta)。
    • 连接句 (て-形): い -> くて: 安くて、おいしい (yasukute, oishii) - 又便宜又好吃。
  • な-形容词 (na-adjectives): 本质是名词，需要 な 来连接另一个名词。
    • 修饰名词: [な-adj] な [名词]: きれいな 花 (kirei na hana) - 漂亮的花。
    • 作谓语: [な-adj] だ: 花 が きれい だ (hana ga kirei da) - 花很漂亮。
    • 否定: だ -> じゃない: きれいじゃない (kirei janai)。
    • 过去式: だ -> だった: きれいだった (kirei datta)。
    • 连接句 (て-形): だ -> で: きれいで、静かだ (kirei de, shizuka da) - 既美丽又安静。

◇ [41-45] 动词时态与否定
#

  • ます-形 (Masu-form): 动词的礼貌体形式，用于正式场合。
    • る-动词: る -> ます: 食べる -> 食べます。
    • う-动词: u 音 -> i 音 + ます: 歩く (aruku) -> 歩きます (arukimasu)。
  • 普通体 (Plain Form) vs 礼貌体 (Polite Form)
    • 课程前期主要教授普通体（词典形、ない形、た形），因其是构成复杂语法的基础。礼貌体只是在句子末尾进行变换。
  • ない-形 (Nai-form / Negative): 动词的否定形。
    • る-动词: る -> ない: 食べる -> 食べない。
    • う-动词: u 音 -> a 音 + ない: 歩く -> 歩かない。
    • 不规则: する -> しない, 来る -> 来ない (konai)。
  • た-形 (Ta-form / Past): 动词的过去式，变形规则与 て-形 完全相同，只需将 て/で 换成 た/だ。
    • 食べて -> 食べた (tabeta)。
    • 歩いて -> 歩いた (aruita)。

★ 第三部分：复合句与高级助词用法 (Lessons 46-70)
#

这部分讲解了如何将简单的句子组合成复杂的逻辑结构，以及一些常用助词的深层含义。

◇ [46-55] 从句与引用
#

  • 核心概念：将句子名词化
    • 日语可以将整个句子作为一个名词，用来修饰另一个名词或作为更大句子的一个部分。
    • 方法1：用 の: [句子] の は [描述]。
      • 私 が ケーキ を 食べた の は 事実だ - 我吃了蛋糕这件事是真的。
    • 方法2：用 こと (koto): 表示抽象的“事情”。
      • 日本語を勉強する こと が 好きだ - 我喜欢学习日语这件事。
  • 引用助词 と (to):
    • 用于直接或间接引用某人的话或想法。
    • [引用内容] と 言う/思う (…to iu/omou) - 说/想“…”
    • 例: 彼 は 「行く」 と 言った (kare wa “iku” to itta) - 他说了“我去”。
  • 用动词修饰名词: 直接将动词普通体放在名词前。
    • 私 が 食べた ケーキ (watashi ga tabeta keeki) - 我吃的那个蛋糕。
    • 本 を 読む 人 (hon o yomu hito) - 看书的人。

◇ [56-65] 条件与原因
#

  • 条件形 (ば, たら, と, なら):
    • と (to): 自然而然的结果。只要A发生，B必然发生。
      • 春 に なると、花 が 咲く (haru ni naru to, hana ga saku) - 一到春天花就会开。
    • たら (tara): 动词 た-形 + ら。最通用的条件形，表示“如果…之后”。
      • 日本 に 行ったら、寿司 を 食べる (Nihon ni ittara, sushi o taberu) - 如果去了日本，就吃寿司。
    • ば (ba): 假设性最强，多用于逻辑推导或普遍真理。
      • 动词: u -> e + ば (行けば)。い-adj: い -> ければ (安ければ)。
      • 安ければ、買う (yasukereba, kau) - 如果便宜的话，就买。
    • なら (nara): 基于对方提供的信息或上下文提出建议或限制。
      • A: “我想去日本。” B: 日本 に 行く なら、京都 が おすすめ だ よ - 如果是去日本的话，我推荐京都。
  • 原因/理由 (から, ので):
    • から (kara): 主观原因，强调说话者的判断和理由。
      • お腹が空いた から、レストラン に 行く - 因为肚子饿了，所以去餐厅。
    • ので (node): 客观原因，语气更委婉、礼貌。
      • 電車 が 遅れた ので、遅刻しました - 因为电车晚点了，所以我迟到了。

◇ [66-70] 给予与接收
#

  • 授受动词:
    • あげる (ageru): 我/我方的人 -> 给别人 (或别人给别人)。
      • 私 が あなた に プレゼント を あげる。
    • くれる (kureru): 别人 -> 给我/我方的人。
      • あなた が 私 に プレゼント を くれる。
    • もらう (morau): 从别人那里“得到”。主语是接收方。
      • 私 が あなた に/から プレゼント を もらう。
  • てあげる, てくれる, てもらう:
    • 将授受关系用于“行为”上，表示为某人做某事。
    • てあげる: 为别人做某事。
    • てくれる: 别人为我做某事。
    • てもらう: 我请别人为我做某事。
    • 例: 友達 が 宿題 を 手伝ってくれた (tomodachi ga shukudai o tetsudatte kureta) - 朋友（为我）帮忙做了作业。

★ 第四部分：进阶语法与表达 (Lessons 71-93)
#

这部分涵盖了更复杂的语法结构，如被动、使役、能力以及各种表达细微语气的句末助词。

◇ [71-80] 被动与使役
#

  • 被动形 (Passive Form):
    • 表示“被…”。
    • る-动词: る -> られる: 食べられる (taberareru) - 被吃。
    • う-动词: u -> a + れる: 書く -> 書かれる (kakakeru) - 被写。
    • 不规则: する -> される, 来る -> 来られる (korareru)。
    • 例: 私 は 先生 に 褒められた (watashi wa sensei ni homerareta) - 我被老师表扬了。
  • 使役形 (Causative Form):
    • 表示“让/使…做…”。
    • る-动词: る -> させる: 食べさせる (tabesaseru) - 让…吃。
    • う-动词: u -> a + せる: 書く -> 書かせる (kakaseru) - 让…写。
    • 不规则: する -> させる, 来る -> 来させる (kosaseru)。
    • 例: 母 は 子供 に 野菜 を 食べさせた (haha wa kodomo ni yasai o tabesaseta) - 妈妈让孩子吃了蔬菜。
  • 使役被动形 (Causative-Passive):
    • 表示“被迫做…”，结合了使役和被动。
    • 例: 毎日、母 に 勉強させられる - 每天都被妈妈逼着学习。

◇ [81-85] 能力与意向
#

  • 可能形 (Potential Form):
    • 表示“能/会做…”。
    • る-动词: る -> られる: 食べられる (taberareru) - 能吃。 (与被动形相同，靠上下文区分)
    • う-动词: u -> e + る: 書く -> 書ける (kakeru) - 能写。
    • 不规则: する -> できる, 来る -> 来られる (korareru)。
    • 助词 が 常用于可能形前，表示能力的对象。
      • 私 は 日本語 が 話せる (watashi wa nihongo ga hanaseru) - 我会说日语。
  • 意向形 (Volitional Form):
    • 表示“打算做…吧/一起做…吧”。
    • る-动词: る -> よう: 食べよう (tabeyou) - 吃吧。
    • う-动词: u -> o + う: 行こう (ikou) - 走吧。
    • 礼貌体是 ましょう (食べましょう, 行きましょう)。

◇ [86-93] 句末助词与其他表达
#

  • 句末助词 (Sentence Final Particles):
    • ね (ne): 寻求同意、确认信息。“…对吧？”
    • よ (yo): 告知对方不知道的信息，带有强调语气。
    • よね (yo ne): 既强调，又寻求确认。
    • か (ka): 疑问。
  • すぎる (sugiru): “太过于…”。
    • 接动词 ます-形 词干或形容词词干。
    • 食べすぎる (tabesugiru) - 吃太多。
    • 高すぎる (takasugiru) - 太贵。
  • にくい (nikui) / やすい (yasui): “难于…” / “易于…”。
    • 同样接动詞 ます-形 词干。
    • このペンは書きやすい (kono pen wa kakiyasui) - 这支笔很好写。
  • 传闻 そうだ (sou da): “听说…”。
    • 接动词或形容词普通体。
    • 雨が降るそうだ (ame ga furu sou da) - 听说要下雨。
  • 样态 そうだ (sou da): “看起来好像…”。
    • 接动词 ます-形 词干或形容词词干。
    • 雨が降りそうだ (ame ga furisou da) - 看起来快要下雨了。

★ 结构化总结
#

这个课程的核心是 解构日语语法，将其还原为几个简单、一致的逻辑模块。

  1. 句子 = 主格 (が) + 谓语: 所有句子的起点。は 是在此基础上的话题化操作。
  2. 助词定义功能: 每个助词 (を, に, で, と, から, まで 等) 都像一个带有特定功能的“车厢”，为句子添加宾语、地点、时间、工具等信息。
  3. 动词/形容词活用是关键: 掌握 て形、ない形、た形 等核心变形，是构建复杂句子（如条件、时态、请求）的基础。
  4. 从普通体开始学习: 课程强调先掌握普通体（词典形），因为它是所有高级语法的基础。礼貌体 (ます) 只是在句尾添加的“外壳”。
  5. 上下文和省略是常态: “隐形主语”的概念贯穿始终，理解这一点是听懂和说出地道日语的关键。

总而言之，“Japanese from scratch” 课程通过一种“由内而外”的方式教学，首先让学习者牢牢掌握最核心的句子骨架和逻辑，然后像搭积木一样，逐步添加更复杂的语法模块。这种方法有助于学习者形成对日语的直觉性理解，而不是死记硬背零散的规则。
