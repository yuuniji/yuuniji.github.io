★教材的特点
#

日本国际交流基金会编写的《いろどり》（Irodori）是一套专为外国人设计的日语教材，旨在帮助学习者在日本的生活和工作中掌握基础的日语沟通能力。
《いろどり》的核心理念是通过沟通与文化理解，促进人与人之间的相互理解。教材不仅注重语言知识的传授，还特别强调在实际生活场景中的应用。它的学习目标基于JF生活日语CanDo标准，帮助学习者在日本的生活中能够应对各种日常场景，如购物、工作、社交等。
《いろどり》的特点：

  1. 增加在日本生活场景中“能够做到”的事情
  2. 掌握AI～A2级别的日语
  3. 基于“JF生活日语Can-do”
  4. 通过沟通和文化理解，促进人与人之间的相互理解
  5. 包含与生活密切相关的各种话题
  6. 使用丰富的音频和真实材料，学习实用的日语
  7. 扎实学习语法和汉字等语言知识
  8. 可以从网站自由下载
  9. 与《まるごと》有松散关联

★教材的结构
#

《IRODORI》系列教材由《入门（A1）》《初级1（A2）》《初级2（A2）》3部分构成。每本教材分为以下几个部分。

练习，听力课文，汉字词汇，语法笔记，日本生活小知识

★教材的使用方法
#

◇每一课的结构
#

《入门》《初级1》《初级2》均由9个话题、18课内容构成。每一课的授课时间约为150分钟至180分钟。但是，由于每一课的练习量不同，所以请根据实际情况调整授课时间。
【话题和本课的标题】
【热身问题】 热身问题可以使学员具体认识到每一课所涉及的主题。其形式包括回顾自己的经历，或者在班级上进行讨论。
【练习】 练习旨在达成Can-do目标，是本教材的核心部分。每一课有3-6个练习。通过练习，掌握实用的日语。
【听力课文】
【汉字词汇】 学习练习中出现的汉字词汇。
【语法笔记】 对每一课中学习的句型或表达进行解说。
【日本生活小知识】 对练习中出现的日本文化或日本的情况进行解说。

◇练习的种类和目标
#

每一课都有听 说 读 写四种练习。虽然每一个练习都是独立的，但在同一课中各个练习间又都存在一定的联系。不同的课，练习量和练习的种类以及排列顺序可能存在区别，但整体而言，说的练习比较多。四种练习的整体目标如下所示。
说 能够就身边的情景进行提问或回答，
能够就自己的事情或身边的事情进行简单的说明。
听 能够在日常会话中理解对方所说的重要内容，
能够从听到的简单新闻或公共广播等内容中获取必要的信息。
读 能够读取日常生活中常见的通知、公共设施上的告示、餐饮店的菜单中的必要信息，
能够阅读使用简明日语撰写的面向外国人的宣传册等并理解其内容。
写 能够填写日常生活中必要的表格，给朋友等发送信息，
就身边发生的事在社交网络上发布简单的信息。

◇所有练习共同的学习方法
#

各项练习均按照以下顺序进行：
➊ 确认Can-do 查看该练习的Can-do，确认将要掌握的内容。
➋ 练习 进行“听”“说”“读”“写”的练习。
➌ Can-do自我评价表 使用教材最后的Can-do自我评价表，对自己是否达成了Cando目标进行评价。
由每一位学习者对自己的Can-do目标达成情况进行检查。
并在评价栏中自由填写练习的感想以及今后要尝试的事等。
Can-do自我评价表可以在完成一项练习后填写，也可以在学完该课后一起填写。
🟢具体做法：
1.首先确认 Can-do 目标
2.预习词汇
3.听/读（对话范例）
4.关注形式 ↓在下划线处填入单词，注意句型和表达方式。
↓思考句型、含义和用法
↓ 看“语法笔记”的讲解进行确认
↓使用“语法练习册”巩固语法。（也可在完成一套完整的 Can-do 活动或一整课后再进行复习）
5.（口语练习：） 在以“说”为目标的活动中，能够熟练使用语法后，可以进行影子练习、角色扮演等口语练习
6.Can-do 检查 请在评论部分分享你对活动的想法以及下一步计划，
自由地写下你想做的事情。

《IRODORI》这套教材的语法学习的关键是首先要输入，然后关注输入的日语语法的形式和含义，并通过自己的思考发现规则的过程。因此，正如下面的例子所示，从语法入手，进行讲解和练习，然后再转向活动，并不是一个好主意。

❌不是很好的范例：

  • 导入语法
  • 用“语法笔记”说明讲解（课堂/预习）
  • 用“语法练习册”练习（课堂/预习）
  • 进入活动
    • 预习词汇
    • 听/读（对话范例）
    • 关注形式

“语法练习册”是为了完成《IRODORI》中的 Can-do 目标而练习语法的辅助资料。只学习“语法练习册”上的知识点是没有意义的，请务必配合《IRODORI》教材使用。为了能够在现 实生活中进行听说，实现真正的交流，《IRODORI》教材中的会话练习是必不可少的。

◇“说”的练习
#

说
➊ 确认Can-do
➋ 词汇的预习 学习练习所需的词汇。根据练习不同，有时可能没有需要预习的词汇。按照以下的顺序进行学习：
1. 边看插图边听录音，并确认意思
2. 听录音，并模仿发音：目的是为了确认发音，而不是记住词汇。
3. 听录音，选择与内容相符的插图：确认在听到录音后能否马上理解意思。目的是帮助学员将含义和表达形式结合起来。
➌ 听对话示例 虽然练习的最终目标（Can-do）是“说”，但为了达成Can-do目标，要先听对话示例，大致理解对话内容。在这一阶段，无需详细理解语法或表达的用法。
根据练习的不同，对话示例分为“在若干不同情景中，听若干较短对话”和“在同一情景中，听较长对话”。原则上，在第一种情况下，课文将不出现在此处，而是作为“听力课文”出现在后面。在第二种情况下，此处将出现课文。两种情况下的学习方法如下所示：
没有对话课文时：
1. 在听录音前，确认是在什么样的情景下关于什么事情的对话
2. 听对话，一边回答问题，一边理解重要的内容
3. 确认🔲中的新词汇或表达的意思后，再听一遍录音，理解更多细节
有对话课文时：
1. 听录音前，确认是在什么样的情景下，谁和谁在说话
2. 先不看课文，只听录音，一边回答问题，一边理解重要的内容
3. 边看课文边听录音，更加详细地理解内容，这时，一起确认🔲中的新词汇或表达的意思
➍ 注意形式 学习为达成Can-do目标所需的句型或表达。
1. 注意句型或表达：在对话示例中，希望学习者注意的句型或表达的部分已留白并添加下划线。通过听录音并将词汇填入空白处，促使学习者关注此处的说法和形（语言形式）。
2. 思考句型或表达的意思及使用方法：通过回答问题❕，思考句型或表达的意思、使用方法。教师先不讲解语法，而是让学习者通过自己思考，发现日语的规则。
➡ 然后，通过阅读“语法笔记”中的例句及解说进行确认。
3. 关注形式，再听一遍对话示例：再听一遍对话示例，确认学过的句型或表达在对话中是如何使用的。
➎说 这是达成练习最终目标（Can-do）的部分。一点一点进行阶段性的练习。
1. 听示范对话：“对话框对话”是为了达成Can-do目标的示范对话。一边看“对话框对话”，一边听录音，确认谈话的过程及表达。
2. 跟读：这是为了能够流畅说话而进行的练习。一开始，先看着“对话框对话”进行跟读。重复几遍后，不看“对话框对话”进行跟读。
3. 练习：这是为了进行“4.自由会话”而做的准备。有的课中可能没有。替换“对话框对话”中的词汇进行练习，或者使用 ➌ 的对话示例的内容进行对话。
4. 自由会话：随便说说自己的情况，或者进行角色扮演。除替换“对话框对话”中的词汇进行对话外，还可以随便说自己想说的话。使用词典或智能手机等查一查必要的词汇吧。
➏ Can-do自我评价表

ℹ️什么是跟读？
“跟读”指的是将听到的日语，立即按照原样模仿的练习方法。不需要全部听完然后重复，而是听到后立即像影子（shadow）一样跟着模仿，小声进行发声练习。

◇“听”的练习
#

练习的学习方法与“说”相似，最终目标（Can-do）是听后能够理解，因 此如果能在听到后理解必要的内容，则为目标达成。

听
➊ 确认Can-do
➋ 词汇的预习 在听录音前，学习需要掌握的词汇。根据练习的不同，可能有也可能没有。
学习方法与“说”的练习相同。
➌ 听 这是达成练习最终目标（Can-do）的部分。
1. 确认设定：听录音前，一边看插图等，一边确认是什么样的情景、为了什么、要听什么。
2. 阶段性理解内容：一边回答问题，一边理解重要内容。“选择与内容相符的插图”“选择关键词”“记录要点”“划〇划×”等，根据练习的不同，要做的内容也不同。没有必要完全明白听到的内容。只要能够回答问题，就算达成了Can-do目标。
3. 确认词汇后再听一遍：为了能够听懂更详细的内容而进行的拓展练习。确认🔲中的新词汇或表达的意思后，再听一遍。有时会出现超出初级水平难度的词汇，因此没有必要全都记住。
➡ 想要进一步确认不明白的地方时，可以查看后面的听力课文。
➍ 注意形式 关注听到并理解的内容中出现的新句型或表达，进行学习。 学习方法与“说”的练习相同。但是，没有新句型或表达时，则没有该环节。
➎ Can-do自我评价表

◇“读”的练习
#

读
➊ 确认Can-do
➋ 读 这是达成练习最终目标（Can-do）的部分。
1. 确认设定：读课文前，确认是什么样的情景、为了什么、要读什么。
2. 阶段性理解内容：
一边回答问题，一边理解必要的内容。先了解概要，例如“写的是关于什么的内容”“必要的信息是写在哪里的”等，然后进一步读取较为详细的信息。即使没有全部看懂，只要能够回答问题，就算达成了Can-do目标。在日本的生活中实际看到的文字，一般是不带标音（注音假名）的。因此，实际生活中不标音的，阅读素材中也没有标音。实际在日本生活时，大家会看到很多不认识的词汇或者不知道读法的汉字。所以，这里假设大家就是在日本生活，首先不要确认“重要的词汇”中的单词含义，也不要使用辞典等查询不认识的词汇，而是以认识的词汇、图片、插图等为线索，边猜边读。
🔖【大切なことば】　确认“重要的词汇”：
“重要的词汇”是从阅读素材中挑选出来的，主要包括初级水平的学习者需要掌握的词汇，以及在日本生活需要知道的词汇。先依靠自己的能力阅读，然后再确认“重要的词汇”的意思，或用辞典等查询汉字的读法或不认识的词汇。然后，可以再读一遍。
3. 加深对阅读内容的理解：回答问题，通过回顾自己的经历，或者与自己国家的情况进行对比，加深理解。在班级中进行讨论，这时 可以使用其他语言。根据练习的不同，可能有也可能没有。
➌ 注意形式 主要涉及阅读素材中初级水平的学习者需要掌握的句型或表达。学习方法与“说”“听”的练习相同。不过，“读”的练习中没有录音。一边看文章，一边在划线处填写词汇。根据练习的不同，可能有也可能没有。
➍ Can-do自我评价表

◇“写”的练习
#

写
➊ 确认Can-do
➋ 写 1. 确认设定：写之前，确认是什么样的情景、为了什么、要写什么。
2. 阅读示例：如果有示例，则先阅读示例，具体了解要写什么样的内容。有时，会直接使用前面的阅读练习的示例。
3. 写：实际生活中，既有填表等需要手写的情况，也有发送信息等使用智能手机或键盘进行输入的情况。因 此，发送信息或者在社交网络发布消息时，尽可能使用智能手机或键盘输入。
4. 就自己写的内容收集反馈：在班级中交换阅读各自写下的内容，进行评论或者回复信息。收集阅读者的反馈，能够在“写”的练习过 程中更好地意识到阅读者的存在，有助于实际的交流。根据练习的不同，可能有也可能没有。
➌ Can-do自我评价表

◇汉字词汇
#

在该环节中，学习者将初级水平的汉字放到词汇中进行练习。这一环节的目标是在日常生活中看到必要的汉字能够理解意思，并在需要时能够使用智能手机或电脑输入汉字。
每一课都将在练习的词汇中挑选10个左右的汉字词汇（《入门》《初级1》《初级2》中学习的汉字共429个）。按照以下的顺序进行学习。

➊ 确认读法和意思
关注汉字的读法，并确认词汇的意思。汉字使用3种字体书写。以便让学习者熟悉各种不同的字体。
➋ 在句子中阅读
阅读包含汉字词汇的句子，确认能否读出、是否明白意思。
➌ 输入
最后，使用自己的智能手机、平板电脑、键盘等输入学过的汉字词汇，确认能否正确输入汉字。这一环节并不要求手写汉字，对汉字感兴趣的学习者，想要练习手写汉字的话，可以自行练习。

◇语法笔记
#

对每一课中学习的句型或表达进行解说。涉及各个练习中“关注形式”的句型或表达。阅读解说后，即可知道“关注形式”中问题❕的答案。
针对每一课中出现的项目，就形（语言形式）、意思以及用于该课中怎样的情景进行了解说。此外，还有本教材的练习中未涉及到的使用方法等拓展信息以及追加的例句。必要时，还会追加活用变形等的汇总列表、类似表达的对比等，帮助学习者整理语法知识。
在课程中，学习者可以先通过练习中的“关注形式”，自行思考句型及表达的使用方法，然后阅读本环节中的解说，也可以由教师使用本环节的内容进行解说。还可以在课程中只进行简单的介绍，然后将解说部分作为作业，留给学习者课后阅读。
此外，本教材中没有以学习语法为目的的练习。如有需要，请使用《MARUGOTO》系列教材中的“理解”（用于提高语言交际能力的教材）等其他教材进行辅助学习。不过，在进行语法练习时，请注意Can-do目标，不要过度偏离 Can-do目标。

在表示词类等时使用以下标记。
N…… 名词
ナA…… ナ形容词
イA…… イ形容词
V…… 动词
S…… 句子
“V-る”表示动词的原形。也包括“読む”“行く”等不以“る”结尾的动词。

◇日本生活小知识
